Launch Arguments

Space-separated Chromium / Electron flags passed to Discord on launch. Changes require a restart.

Only use arguments you understand. Incorrect arguments may cause instability or security issues.
Common examples:
  • --force_high_performance_gpu — prefer the discrete GPU on hybrid systems
  • --ozone-platform-hint=auto — let Electron pick Wayland vs X11 (override auto-detect)
  • --enable-features=VaapiVideoDecodeLinuxGL,AcceleratedVideoDecodeLinuxGL — VA-API hardware video decode (Linux)
  • --disable-gpu — disable hardware acceleration
